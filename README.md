
# ☕ Cafe Management System

A web-based Cafe Management System developed using **HTML, CSS, JavaScript, PHP, and MySQL**.  
The system includes an **Admin Panel** to efficiently manage cafe operations such as menu management, orders, users, and sales tracking.

---

## 📌 Project Overview

The Cafe Management System is designed to automate daily cafe operations and improve efficiency.  
It allows admins to manage menu items, handle customer orders, and monitor sales through a simple and user-friendly interface.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Secure Admin Login System
- Add, update, and delete menu items


### 🍔 User Side
- Browse cafe menu
- View item details
- Place orders
- Simple and interactive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP / Apache Server

---

## ⚙️ Installation & Setup

1. Clone or download the project
```bash
git clone https://github.com/your-username/cafe-management-system.git
````

2. Move project to `htdocs` folder (XAMPP)

3. Start Apache & MySQL using XAMPP

4. Create database in phpMyAdmin

5. Import the SQL file provided in the project

6. Run the project in browser

```
http://localhost/cafe-management-system/
```
